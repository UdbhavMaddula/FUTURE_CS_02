# Phishing Email Detection & Awareness System

## Overview

This project was completed as part of the Future Interns Cyber Security Internship Program (Task 2). The objective was to analyze phishing emails, identify common attack indicators, classify email risks, and create user awareness guidelines to help individuals and organizations defend against phishing attacks.

The project follows the workflow of a Security Operations Center (SOC) Analyst by examining suspicious emails, investigating email headers, analyzing sender domains, identifying Indicators of Compromise (IOCs), and correlating findings with threat intelligence sources.

## Objectives

* Analyze phishing email samples
* Identify phishing indicators and attack techniques
* Classify email risks (Safe, Suspicious, or Phishing)
* Investigate email headers and sender reputation
* Document findings in a professional incident report
* Create phishing awareness and prevention guidelines

## Tools Used

* TryHackMe Email Security Simulator
* Email Header Analysis
* Cisco Talos Intelligence
* Web Browser Investigation Tools
* Microsoft Word
* GitHub

## Analysis Approach

1. Collected phishing email samples.
2. Examined sender information and email content.
3. Identified suspicious domains, links, and attachments.
4. Performed reputation checks on domains and IP addresses.
5. Classified each email based on risk level.
6. Documented findings with supporting evidence.
7. Created user awareness recommendations and prevention guidelines.

## Key Findings

* Multiple phishing techniques were identified, including inheritance scams, invoice-themed phishing, prize scams, credential harvesting attempts, and malicious URL campaigns.
* Common phishing indicators included urgency, suspicious sender domains, social engineering tactics, malicious attachments, and shortened URLs.
* Firewall and threat intelligence logs confirmed the blocking of a malicious external URL, preventing potential compromise.

## Conclusion

This project demonstrates practical phishing detection and analysis skills, including email investigation, threat identification, IOC analysis, and cybersecurity awareness reporting. The findings highlight the importance of user education, email security controls, and proactive threat monitoring in defending against phishing attacks.
